## 型
- Haskellにおいて**型とは関連する値の集合**のこと

## 型安全
- 式の評価前に型検査がなされるため、Haskellのプログラムは型安全であることが保証されている

## 型変数
- 型変数は任意の型を表す
- 型変数は必ず小文字で始まる

## 多相型
- １つ以上の型変数を含む型や、1つ以上の型変数を含む式を多相型と呼ぶ

## 型の表記
- 以下の表記により「vは型Tの値」であると表現される

``
v::T
``

## 関数型

- 関数も型をもつ
- 関数notは型Bool->Boolの値

```
Prelude> :type not
not :: Bool -> Bool
```
